# Угадай число - Веб приложение с REST API

Веб-версия классической игры "Угадай число" с клиент-серверной архитектурой и REST API.

## Запуск приложения

### Требования
- PHP 7.4 или выше
- Веб-браузер с поддержкой JavaScript

### Инструкция по запуску

1. **Перейдите в корневую директорию проекта**:
   ```bash
   cd Task08
   ```

2. **Запустите встроенный PHP сервер**:
   ```bash
   php -S localhost:3000 -t public
   ```

3. **Откройте приложение в браузере**:
   ```
   http://localhost:3000
   ```

   Или напрямую:
   ```
   http://localhost:3000/index.html
   ```

## REST API Endpoints

### Получить список всех игр
```
GET /games
```

### Получить данные конкретной игры
```
GET /games/{id}
```

### Создать новую игру
```
POST /games
```
Тело запроса:
```json
{
  "playerName": "string",
  "maxNumber": number,
  "maxAttempts": number,
  "secretNumber": number
}
```

### Сохранить ход игры
```
POST /step/{id}
```
Тело запроса:
```json
{
  "attemptNumber": number,
  "guess": number,
  "result": "string"
}
```

## Функциональность

- Создание новой игры с настройками
- Сохранение ходов в базе данных
- Просмотр истории игр
- Воспроизведение завершенных игр
- Продолжение незавершенных игр
- Статистика по игрокам
