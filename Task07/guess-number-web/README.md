# Угадай число - Веб версия

Веб-версия классической игры "Угадай число", реализованная на JavaScript с использованием IndexedDB для хранения данных.

## Описание игры

Компьютер загадывает число в заданном диапазоне, а игрок должен угадать его за ограниченное количество попыток. После каждой попытки игрок получает подсказку: "больше" или "меньше".

## Возможности

### Основной игровой процесс
- Настройка параметров игры (имя игрока, диапазон чисел, количество попыток)
- Интуитивно понятный интерфейс с визуальными подсказками
- Отслеживание прогресса в реальном времени

### Работа с базой данных
- **Автоматическое сохранение** всех игровых сессий в IndexedDB
- **Просмотр истории** всех сыгранных партий с фильтрацией по статусу (все/победы/поражения)
- **Воспроизведение игр** - возможность просмотреть ход любой завершенной игры
- **Статистика игроков** - детальная статистика по всем игрокам:
    - Общее количество игр
    - Количество побед и поражений
    - Процент побед
    - Среднее/минимальное/максимальное количество попыток для победы

### Управление данными
- Все данные хранятся локально в браузере
- Быстрый доступ к истории игр
- Возможность просмотра детальной информации по каждой игре

## Технологии

- **HTML5/CSS3** - структура и стилизация

## Структура базы данных

### Хранилище `games`:
- `id` - уникальный идентификатор игры
- `playerName` - имя игрока
- `secretNumber` - загаданное число
- `maxNumber` - максимальное число в диапазоне
- `maxAttempts` - максимальное количество попыток
- `isCompleted` - завершена ли игра
- `isWon` - победа в игре
- `attemptsCount` - количество сделанных попыток
- `startTime`, `endTime` - время начала и окончания игры

### Хранилище `attempts`:
- `id` - уникальный идентификатор попытки
- `gameId` - ссылка на игру
- `attemptNumber` - номер попытки
- `guess` - предложенное число
- `result` - результат попытки (win/greater/less)
- `attemptTime` - время попытки
